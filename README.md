# Income Prediction using Machine Learning

## Overview
This project builds machine learning models to predict whether an individual earns more than $50K per year.

## Dataset
UCI Adult Census dataset.

## Models
- Logistic Regression
- Decision Trees
- Random Forest

## Results
Random Forest achieved the best performance:

Accuracy: 85.54%
Precision: 75.32%
Recall: 61.19%

## Tools
Python / R
Scikit-learn
Pandas
